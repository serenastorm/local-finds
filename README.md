# Local Finds

## Compile TS to JS

Loop through all `.ts` files in the `scripts` folder and generates `.js` files:

```bash
npm run typescript
```

## Minify JS files

Minify the generated `.js` files and save them to the `lib` folder:

```bash
npm run minify
```

## Build project

Compile and minify all `.js` files in the `scripts` folder:

```bash
npm run build
```
